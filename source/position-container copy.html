Posicionamiento contenedores Orden 1

Contenedor static (dad)

Soy un elemento P (son), dentro de un contenedor Static. Mi contenido puede ser centrado de forma horizontal si al estilo de mi caja contenedora (dad) se le agrega la propiedad text-align: center; o incluso a mi propio estilo.

La siguiente imagen tambien es un elemento inline, se centra de forma horizontal con la propiedad text-align:center;

Soy en div dentro de un contenedor Static. Para centrarme de forma horizontal en mi estilo se incluyeron las siguientes propiedades: position: relative; y margin: 10px auto;
